# Heterogeneity-assessment-through-meta-analysis_Janmanjaya
Aim of the project: Assessment of Heterogeneity in Gene Expression Datasets Through Meta-Analysis
Data type: Microarray Gene Expression Datasets
Packages used: Refer to scripts

About the scripts: The gene expression data was processed using the scripts shared. Below are the details of the analysis performed using the R scripts.

Quantile normalization: Gene expression microarray data was quantile normalized.
Batch correction: The datasets were merged and batch corrected.
DGEA and Meta-analysis: DGEA and Meta-analysis of the results was performed.
Gene filtering: Genes were selected based on a certain criteria.
Pseudogene synthesis: The selected genes were used to synthesize pseudogenes.
Meta-analysis: Pseudogene was used to assess dataset level heterogeneity.
